Check if Element Is First in a Set

Usage

is.first_in_set(x)


is.last_in_set(x)

Arguments

x
A vector of elements.

Check if Element Is First in a Set

Check if Element Is First in a Set

Value

Returns a logical vector equal in length to x.

Description

is.first_in_set - Logical check of each element in a vector to determine if an element is first in set.

is.last_in_set - The (semantical) "inverse" to is.first_in_set.

Examples

set.seed(3)
(x <- sample(1:3, 9, TRUE))



[1] 1 3 2 1 2 2 1 1 2



is.first_in_set(x)



[1]  TRUE  TRUE  TRUE  TRUE  TRUE FALSE  TRUE FALSE  TRUE



is.last_in_set(x)



[1]  TRUE  TRUE  TRUE  TRUE FALSE  TRUE FALSE  TRUE  TRUE
