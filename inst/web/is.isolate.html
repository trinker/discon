Check if Element Is Isolated

Usage

is.isolate(x)


is.repeat(x)

Arguments

x
A vector of elements.

Check if Element Is Isolated

Check if Element Is Isolated

Value

Returns a logical vector equal in length to x.

Description

is.isolate - Logical check of each element in a vector to determine if an element is isolated, that is not repeated.

is.repeat - The (semantical) "inverse" to is.isolate.

Examples

set.seed(3)
(x <- sample(1:3, 9, TRUE))



[1] 1 3 2 1 2 2 1 1 2



is.isolate(x)



[1]  TRUE  TRUE  TRUE  TRUE FALSE FALSE FALSE FALSE  TRUE



is.repeat(x)



[1] FALSE FALSE FALSE FALSE  TRUE  TRUE  TRUE  TRUE FALSE
